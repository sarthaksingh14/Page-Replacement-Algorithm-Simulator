How Each Algorithm Works

📦

FIFO

First In, First Out — the simplest strategy. The page that has been in memory the longest is replaced first, regardless of how recently or frequently it was used.

FIRST IN FIRST OUT
🔁

LRU

Least Recently Used — replaces the page that hasn't been accessed for the longest time. Approximates the optimal algorithm using past usage patterns.

LEAST RECENTLY USED
🎯

Optimal

The theoretical best — replaces the page that will not be used for the longest time in the future. Serves as a benchmark; impossible to implement in real OS without future knowledge.
